# Sathi 🌟
### Your emotionally aware, multilingual 2AM AI best friend

Sathi is a real-time voice AI companion built with Gemini Live API that detects your emotional state, switches languages automatically, and responds like a caring human friend — not a robot.

## Features
- 🎙️ Real-time voice conversations powered by Gemini Live API
- 💜 Emotionally aware — detects stress/frustration and responds with empathy first
- 🌍 Multilingual — automatically detects and responds in your language
- 5 modes: Vibe, Slow Mode, Quiz Me, Debate, Classroom

## Tech Stack
- Frontend: HTML, CSS, JavaScript (WebSockets)
- Backend: Python FastAPI
- AI: Google Gemini Live API
- Deployment: Google Cloud Run

## Setup Instructions
1. Clone the repo
2. Create `.env` file with your `GEMINI_API_KEY`
3. Install dependencies: `pip install -r requirements.txt`
4. Run: `uvicorn main:app --reload`
5. Open `http://localhost:8000`
